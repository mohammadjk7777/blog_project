# پروژه وبلاگ شخصی (Blog Project)

این پروژه به عنوان پروژه نهایی درس **طراحی مبتنی بر وب** برای **استاد رمضانیان** توسط **محمد جعفر کریمی** طراحی و پیاده‌سازی شده است.

## معرفی پروژه
این وب‌سایت یک سیستم مدیریت محتوای ساده (CMS) برای وبلاگ‌نویسی است که با استفاده از زبان برنامه‌نویسی PHP و دیتابیس MySQL توسعه یافته است. هدف از این پروژه، پیاده‌سازی عملی مفاهیم طراحی وب و تعامل با پایگاه داده بوده است.

## ویژگی‌های پروژه
- **بخش کاربری:** مشاهده پست‌ها، نمایش جزئیات هر پست، ثبت دیدگاه.
- **بخش مدیریت:** افزودن پست جدید، ویرایش و حذف پست‌ها، مدیریت دسته‌بندی‌ها و کاربران.
- **ثبت‌نام و ورود:** سیستم مدیریت کاربران با امنیت پایه.

## تکنولوژی‌های استفاده شده
- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML5, CSS3

## نحوه اجرا
1. مخزن (Repository) را کلون کنید یا فایل‌ها را دانلود کنید.
2. فایل `includes/blog_project.sql` را در دیتابیس MySQL خود (از طریق phpMyAdmin) ایمپورت کنید.
3. اطلاعات اتصال به دیتابیس را در فایل `includes/db.php` بر اساس تنظیمات سرور محلی خود (مانند WampServer یا XAMPP) تنظیم کنید.
4. پروژه را در پوشه `www`  قرار داده و از طریق مرورگر اجرا کنید.
